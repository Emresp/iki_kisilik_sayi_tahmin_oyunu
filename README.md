İki kişinin rekabet edebileceği şekilde tasarlanan oyunlaştırılmış sayi tahmini

Oyuna başlayacağınız zman iki oyuncununda isimlerini ister

ilk girilen isim oyuna başlar

ilk kişinin oyunu bitince kaç denemede bulduğu ekrana yazılır. Ardından ikinci oyuncunun oyunu başlatması için girdi beklenir

İkinci oyuncu doğru girdiyi verdiği zaman ikinci oyuncununda oyunu başlar

İkinci oyuncunun oyunu bitince ekrana kaç defada bildiği yazdırılır

son olarak iki oyuncunun kıyaslaması yapılır ve tablo halinde oyunculara gösterilir
